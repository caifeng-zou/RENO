# RENO: Reciprocity-Enforced Neural Operator

## Introduction
This repository provides code for the paper [Enforcing Reciprocity in Operator Learning for Seismic Wave Propagation](http://arxiv.org/abs/2602.11631).

## File Description
- **code**: 
    - **util**: Auxiliary classes and functions
    - **modules**: RENO modules
    - train.ipynb: Code for training a RENO
- **data**: A small amount of training data for running train.ipynb
- **data_generation**: Code for generating training data with [Deepwave](https://ausargeo.com/deepwave/)
- **model**: Trained RENO
